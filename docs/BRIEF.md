# Creative Brief

## SudutKode — Landing Page

**Version:** 1.0.0
**Date:** 2026

---

## 1. Project Overview

Building an official landing page for **SudutKode**, a new software house founded by a group of professionals. This landing page is SudutKode's digital face — where potential clients learn who we are, what we offer, and what products we've created.

---

## 2. Creative Objectives

- Create a strong, professional, and modern first impression
- Reflect SudutKode's technical quality through the page itself
- Make potential clients confident to contact us
- Interest potential UangKu users to download the app

---

## 3. Target Audience

**Primary:** Small to medium business owners who need digital solutions — websites, apps, or technology consulting.

**Secondary:** Individuals looking for a personal finance app (potential UangKu users).

---

## 4. Core Message

> _SudutKode builds digital products that truly work — with a solid team, structured approach, and attention to detail._

This message should come through in:

- Quality of page visuals (clean, not cluttered)
- Copy that is to-the-point and not excessive
- Real product displays (actual screenshots, not empty mockups)

---

## 5. Tone & Voice

| Dimension      | Direction                                                            |
| -------------- | -------------------------------------------------------------------- |
| Language tone  | Professional but not stiff — like talking with a competent colleague |
| Visual style   | Minimalist, dark, whitespace-heavy                                   |
| Reference feel | Linear, Resend, Clerk                                                |
| What to avoid  | Too formal, too many words, generic illustrations, colorful design   |

---

## 6. Visual Identity

### Brand Name

**SudutKode** — written as one word, capital S and K

### Font

**Montserrat** — used throughout the entire page including brand name

| Usage              | Weight         |
| ------------------ | -------------- |
| Brand name / logo  | Semibold (600) |
| Section heading    | Semibold (600) |
| Sub-heading        | Medium (500)   |
| Body / description | Regular (400)  |

### Colors

Currently **monochrome** — black, white, and gray. Brand accent color not final yet, will be added in the next iteration.

| Token          | Value     | Function             |
| -------------- | --------- | -------------------- |
| Background     | `#0a0a0a` | Main page background |
| Surface        | `#111111` | Alternate sections   |
| Card           | `#161616` | Card / panel         |
| Border         | `#222222` | Separator lines      |
| Main text      | `#F5F5F5` | Headings, labels     |
| Secondary text | `#A0A0A0` | Body, description    |
| Muted text     | `#555555` | Placeholder, hints   |
| CTA            | `#FFFFFF` | Primary button       |

### Theme

Permanent dark mode. No light/dark theme toggle.

---

## 7. Page Structure

| Section  | Function                                     |
| -------- | -------------------------------------------- |
| Navbar   | Navigation + "Contact Us" CTA                |
| Hero     | First impression — tagline + two CTAs        |
| About    | Brief SudutKode story                        |
| Services | 4 main services in card grid                 |
| Products | Product showcase (UangKu + upcoming product) |
| Team     | Team member photos and names                 |
| Contact  | Contact channel with SudutKode               |
| Footer   | Closing — navigation + social + copyright    |

---

## 8. Required Assets

| Asset                 | Format                       | Description                        |
| --------------------- | ---------------------------- | ---------------------------------- |
| SudutKode logo / icon | SVG or transparent PNG       | For navbar and favicon             |
| UangKu screenshots    | PNG, portrait, min. 3 images | For Product section carousel       |
| Team member photos    | JPG, square, min. 400×400px  | For Team section, consistent style |
| Favicon               | ICO or PNG 32×32             | Replace `assets/icon.ico`          |

---

## 9. Do's and Don'ts

### Do's

- Lots of whitespace — that's what makes it feel premium
- Typography as main visual element
- Thin borders as decorative elements
- Subtle animations on scroll (if available in template)

### Don'ts

- Background gradients or noise textures
- Generic illustrations / stock images
- Too many colors
- Unnecessary long text
- Dark/light mode toggle

---

## 10. Visual References

The following websites are used as reference for vibe and aesthetics:

- **Linear** (linear.app) — strong typography, dark, very minimalist
- **Resend** (resend.com) — pure black background, emerald accent, clean layout
- **Clerk** (clerk.com) — premium feel, indigo accent, abundant whitespace
